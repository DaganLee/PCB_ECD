# 串口通信控制软件

这是一个基于Qt开发的串口通信控制软件，主要用于与下位机设备进行通信，实现设备连接、电压控制、开关机等操作。

## 功能特性

- 自动检测可用串口设备
- 支持串口连接与断开
- 提供电压控制功能（V1和V2电压调节）
- 支持设备开机和关机操作
- 实时显示通信日志
- 自动滚动日志输出

## 主要组件

1. **DeviceController** - 设备控制核心类，处理与设备的通信
2. **SerialPortManager** - 串口管理类，负责检测和监控串口设备
3. **SerialPortService** - 串口服务类，实现底层串口通信功能
4. **Widget** - 主界面，提供用户交互界面

## 使用说明

1. 打开软件后，会自动检测可用串口
2. 选择串口并点击"打开串口"按钮连接设备
3. 使用"开机"按钮启动设备
4. 在V1和V2区域选择电压值，点击"输出"按钮控制电压
5. 使用"关机"按钮关闭设备

## 开发环境

- Qt 5.x 或 Qt 6.x
- C++11 标准
- QWidgets 模块

## 依赖项

- Qt SerialPort 模块
- Qt Widgets 模块

## 构建说明

1. 安装Qt开发环境
2. 打开项目文件(test.pro)
3. 构建并运行项目

## 项目结构

```
├── DeviceController.cpp/h    # 设备控制核心逻辑
├── SerialPortManager.cpp/h   # 串口检测与管理
├── SerialPortService.cpp/h   # 串口通信服务
├── DeviceProtocol.h          # 通信协议定义
├── widget.cpp/h              # 主界面逻辑
└── widget.ui                 # 主界面设计文件
```
